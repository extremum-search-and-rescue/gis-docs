## Какой браузер лучше подойдёт?
- на Windows, macOS или Android - **Яндекс Браузер** или **Google Chrome**,
- на iPhone и iPad - не имеет значения, все браузеры по сути **Safari**,
- **Internet Explorer** не будет работать совсем.
